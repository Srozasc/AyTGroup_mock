# AyT Group - Maqueta HTML/CSS

## Objetivo
Crear una home institucional/comercial estática para AyT Group que represente ingeniería, salud integral e inclusión marcial en una sola experiencia visual.

## Enfoque visual
- Estilo mixto: hero editorial, secciones en tarjetas y una franja clara para precios y CTA.
- Paleta principal: azul marino y verde oliva.
- Recursos visuales: placeholders para logos e imagen del director, sin depender de assets finales.

## Estructura
- Hero con slogan, propuesta de valor y CTA principales.
- Identidad institucional con el perfil de Sebastián Atal y sus cuatro pilares de autoridad.
- Tres pilares estratégicos: ingeniería, club deportivo/inclusivo y salud integral.
- Bloque destacado para FENAMDI Chile.
- Pricing cards para membresías y valores.
- Cierre con CTA de contacto/conversión.

## Resultado esperado
Una maqueta responsive, clara y lista para revisión visual o para pasar después a una implementación con contenido real, formularios o CMS.
